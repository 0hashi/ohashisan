Reddirt.lab is my demo/test lab running on VMWare ESXi, incorporating Windows and Linux Servers. A couple Microsoft SQL Server databases house application data, and MySQL is the database backend of the playground. Additionally, it includes a couple of Cisco routers, a NETGEAR switch, and mindless hub. Naturally, there's always a laptop (at least one) connected via either wired or wireless, and there are the typical array of cellphones and ubiquitous IoT devices found in most households nowadays.
